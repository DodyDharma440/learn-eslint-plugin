# Get Started

## Quick Start

:::info Info

Make sure [ESLint module](https://nuxt.com/modules/eslint) in your Nuxt.js is already added.

:::


### Install the package

```bash
npm install -D @dodidharma/eslint-config-test
```

or with yarn

```bash
yarn add -D @dodidharma/eslint-config-test
```

### Import configuration

```ts title="eslint.config.mjs"
...
import testConfig from "@dodidharma/eslint-config-test/flat";
...

export default withNuxt([
  testConfig // add new testConfig array item
]);
```

Congrats! the ESLint Plugin Test is installed on your Nuxt.js app! We suggest to restart your VSCode ESLint server after add the configuration
